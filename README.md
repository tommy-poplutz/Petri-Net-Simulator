# Petri Net Simulator
Java-based Petri net simulator for modeling and analyzing dynamic systems.

## Overview
This simulator allows the representation of places, transitions and edges, as well as the simulation of the token flow within the network.

## Technologies
- Java

## Features
- Representation of Petri net components
- Simulation of token-based behavior
- Analysis of basic dynamic processes

## Structure
- `Main.java` – program entry point
- `Net.java` – main logic of the Petri net
- `Place.java` – representation of places
- `Transition.java` – representation of transitions
- `Edge.java`, `Node.java` – structural components
- `Tree.java`, `TreeNode.java` – additional logic for state handling
- `Glycolyse_Pentose_Pathway.txt` – example data for testing the program

## Team
This project was developed as part of a team project (3 members).

## Purpose
The goal of this project was to gain practical experience in modeling dynamic systems and implementing simulations using object-oriented programming.
